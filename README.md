This is basically dozer but system wide and if you move it puts any of the punishments you chose, the default one is pc shutdown

## stuff it does
It has 3 forms of punishments, with the version uploaded rn theres shutdown , close window you were previously using, shutdown pc completely or harmless
It has an ui openable with f6, shutdown everything with f5

you will be able to modify the chances and cooldown too, with my preset being recommended for challenges, i think
